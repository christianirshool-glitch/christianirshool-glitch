### 🛡️ Malware Detection — End-to-End ML Pipeline
> Cybersecurity · Binary Classification · XGBoost · SHAP · 500K records

Full ML pipeline to predict malware infection probability on Windows machines. Dataset of 500,000 records with 84 features — expanded to 114 through feature engineering (version tracking, hardware profiling, security signals). XGBoost final model with SHAP-based interpretability revealing SmartScreen as the strongest predictor.

`XGBoost` `LightGBM` `SHAP` `Feature Engineering` `Production Pipeline`

`F1-Score: 0.633 (val)`

🔗 [View project](https://github.com/christianirshool-glitch/My-projects/tree/main/Detecci%C3%B3n_de_malware)

### 🚗 BMW Used Car Price Prediction
> Regression · Random Forest · Custom Imputation · ~5K records

Predictive model to estimate second-hand BMW prices from vehicle characteristics. Built a custom similarity-based imputation function to handle missing values, followed by extensive EDA (Pearson, Spearman, Cramér's V) and a Random Forest regressor achieving R² ≈ 0.91.

`Random Forest` `Regression` `Custom Imputation` `EDA`

`R²: 0.91`

🔗 [View project](https://github.com/christianirshool-glitch/My-projects/tree/main/proyecto_BMW)

### 🌸 Iris Classification — Supervised vs. Unsupervised
> SVM · K-Means · Comparative Analysis

Comparative study between SVM (supervised) and K-Means (unsupervised) on the classic Iris dataset. SVM achieved 100% test accuracy with 98% cross-validation. K-Means without scaling outperformed the scaled version (ARI: 0.72 vs 0.43), demonstrating that preprocessing can hurt when features are already well-scaled.

`SVM` `K-Means` `ARI` `Silhouette Score` `Comparative ML`

`SVM Accuracy: 100% (test) · 98% (CV)`

🔗 [View project](https://github.com/christianirshool-glitch/My-projects/tree/main/Proyecto_Flores_de_Iris)
